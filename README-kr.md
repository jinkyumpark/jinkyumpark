  - 기술의 원리를 이해하는걸 좋아하고, 컴퓨터의 이론적 측면에도 관심이 많아 직무와 관련되지 않더라도 여러 CS 강의를 유튜브에서 보거나 기술서적을 읽는 것을 좋아합니다.
  - 어떤 것을 하더라도 완전히 몰입해 의미있는 것을 이루는 것에서 동기부여를 얻습니다.
